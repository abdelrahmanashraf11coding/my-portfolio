**Default Password:** `admin123` *(change after first login!)*

### Admin Features:
- ➕ **Add** new projects with full details
- ✏️ **Edit** any existing project
- 🗑️ **Delete** with confirmation modal
- 🙈 **Hide/Show** projects without deleting
- ⠿ **Drag & Drop** to reorder projects
- 🖼️ **Media Upload** — image file, image URL, or video URL
- 🌐 **Bilingual** — Arabic & English titles/descriptions
- 📈 **Analytics** — visit counter & storage stats
- 🔐 **Change Password** from Settings tab
- 🔄 **Reset** to default projects

```bash

git clone https://github.com/abdelrahmanashraf11coding/Portfolio.git

# 2. Go to repo Settings → Pages
# 3. Source: Deploy from branch → main → / (root)
# 4. Save → wait 1-2 minutes

# Your site will be live at:
# https://abdelrahmanashraf11coding.github.io/Portfolio
```
